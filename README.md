# Credit Card Customer & Transaction Analysis Dashboard

## Tools Used

MySQL, Excel, Power BI, Power Query, DAX, Data Transformation, Dashboard Development

## Project Overview

1. Built interactive Credit Card Customer and Transaction Analysis dashboards using MySQL, Excel, Power BI, Power Query, and DAX to analyze customer behavior and transaction trends.

2. Developed KPI dashboards to monitor key business metrics, including Revenue (55.3M), Interest Earned (8M), Customer Satisfaction Score (3.19/5), and Transaction Volume.

3. Created DAX measures and calculated columns for Age Groups, Income Groups, Revenue Calculations, Week Numbers, and Week-over-Week Revenue Analysis.

4. Imported, cleaned, and transformed customer and transaction data from MySQL into Power BI using Power Query to support reporting and analytics.

5. Designed interactive visualizations with filters, slicers, and drill-down capabilities to enable dynamic analysis and data-driven decision-making.

6. Strengthened skills in Data Analytics, Data Modeling, Data Visualization, Power BI, SQL, DAX, Business Intelligence, and Dashboard Development.
